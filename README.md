Minecraft But Sh*t
A questionable recreation of Minecraft. Emphasis on "questionable."
This is my attempt at making Minecraft in Unity with C#. It's called "But Sh*t" because, well... it's pretty Sh*t.


Features:
- Procedural terrain generation with Perlin noise
- Block breaking system (with a cracking animations which is not used)
- First-person movement and camera controls
- Different block types (grass, dirt, stone, sand, water)
- Trees
- Day/night cycle
- Water blocks with animation
- A torch you can hold
- Crosshair

Future work:
- Block placing (ran out of time)
- Inventory system
- particals

Controls
- WASD - Move around
- Mouse - Look around
- Space - Jump
- Left Click - Break blocks
- Shift - Walk slowly
- (R) Restart game
- Escape - Unlock cursor

Made for educational purposes. Mojang, please don't look at this.

## Setup Instructions

### Adding Your Poster:
1. Create an `images` folder in your repository
2. Save your poster as `poster.png` in that folder
3. The markdown `![Minecraft But It's Shit Poster](images/poster.png)` will display it

### Adding Your Video:
Since GitHub doesn't play videos directly in markdown, you have a few options:

1. **Upload to YouTube** (recommended)
   - Upload your video to YouTube
   - Copy the URL
   - Use format: `[üéÆ Watch Gameplay Video](YOUR_YOUTUBE_URL)`

2. **Store in repository**
   - Create a `videos` folder
   - Add your video file
   - Link to it: `[üéÆ Watch Gameplay Video](videos/gameplay.mp4)`
   - Users will need to download it to watch

3. **Convert to GIF** (shows inline but lower quality)
   - Use a tool like ezgif.com to convert video to GIF
